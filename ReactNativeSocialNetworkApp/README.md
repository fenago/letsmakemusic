## Prerequisites

Before you begin, ensure you have the following installed:

1. **Node.js** (version 18 or higher)  
   - Download and install from [Node.js official website](https://nodejs.org/).

2. **Yarn** (version 4.4.1 or higher)  
   - Install Yarn globally using npm:  
     ```bash
     npm install -g yarn
     ```

3. **React Native CLI**  
   - Install the React Native CLI globally:  
     ```bash
     npm install -g react-native-cli
     ```

4. **Xcode** (for iOS development)  
   - Install Xcode from the Mac App Store.  
   - Set up the iOS simulator.

5. **Firebase CLI** (optional, for Firebase functions)  
   - Install Firebase CLI globally:  
     ```bash
     npm install -g firebase-tools
     ```

---

## Getting Started

1. **Install dependencies**
```bash
yarn install
```

2. **Run the app on Android**
- Start the Android emulator or connect a physical device.
- Run the app:
```bash
yarn android
```
3. **Run the app on iOS**

- Navigate to ios directory and install the pod dependencies.
```bash
pod install
```
- Select a simulator and run the command:
```bash
yarn ios
```
